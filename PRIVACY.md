# Privacy

PureQR is designed to avoid collecting data.

- QR code content is processed locally in the browser.
- Uploaded logo images are read locally in the browser.
- Generated SVG and PNG files are created locally in the browser.
- The app does not include analytics, cookies, accounts, ads, or tracking code.
- The app does not provide a URL shortener and does not redirect QR code scans.

Hosting providers may process ordinary HTTP request metadata, such as IP
address and user agent, when serving the static website files.
